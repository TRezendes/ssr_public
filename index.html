This is Simple Summer Reading, a web app for public library summer reading programs. It is also my master's degree final project.

I invite you to make an account, play with the site, have a look around. If anything doesn't work right, you can let me know at reports@simplesummerreading.com.

A Few Things You Should Know

  • This is Pre-Alpha Software
    • It's not ready for actual use.
    • It works, but it's missing lots of key features.
  • This Site is Under Active Development
    • I reserve the right to change absolutely anything about it at absolutely any time.
    • Don't count on it to work the same way (or at all) from one day to the next.
    • If I need to wipe the entire database and rebuild it from scratch, I will. Without warning.
  • The Privacy Policy
    • There isn't one.
    • The site uses https, passwords are salted & hashed using the best-in-class Argon2 algorithm, but the rest of the database is in plain-text.
    • I use all best practices for securing access to the backend, but—for now—nothing is encrypted in place.
    • Don't put any sensitive information into this system. Regardless of anything else, at a minimum, I have access to all of the data in the database.

I poured a lot of time, energy, and emotion into building this app. I mean to continue doing so until it is a viable tool that can make life easier for library staff and patrons. In the mean time, have fun playing with it. I hope you like it.
